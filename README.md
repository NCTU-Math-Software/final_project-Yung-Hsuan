我做的是project 17,有兩個程式：squarnew.m和squar2.m。    
程式squarnew.m是造出n×n矩陣：如果n是奇數，就造出由1從中間向右順時針依序寫下數字的矩陣，否則就造出由1從偏左上角向右順時針依序寫下數字的矩陣，以及用image顯示質數點為黑色，合數點為白色的矩陣樣子。程式squar2.m是算出n*n矩陣在不造出矩陣的情況下，算出對角線和反對角線的總和。  

程式squarnew.m:

例如：
 
input：3

output:顯示

7 8 9
 
6 1 2
  
5 4 3
 
和image出來的圖

我先造出一個元素都是0的矩陣A，判斷輸入的值n是奇數還是偶數後，觀察矩陣四邊從外層到內層之間的關係，並用for迴圈依序從最外層的下面，右邊，上面，左邊寫進去到內層，來製造這n*n矩陣。
然後用mod指令來判斷每一個位子的點是否是質數
(判斷質數的部分有上網查資料:https://www.itread01.com/content/1549230666.html  :
目的：用matlab編寫一個可以判斷質數的函式。

 1)若是質數，返回1；

 2）若不是，返回0。

程式碼：

    function s = prime_22(n)
    for i = 2 : n-1
        if mod(n,i)==0
            s = 0;   % s = 0 表示為合數
            break; 
        else

            s = 1;   % s = 1 表示為質數
        end
    end   
    end


)。   


如果那個點是質數，就在最先造出的全零矩陣X裡的那個位子設為2，否則就設為1。
最後，定義map:[0 0 0]代表RGB都是0，表示黑色；[1 1 1]代表RGB都是1，表示白色，用colormap和image顯示著色後矩陣的樣子。  


程式squar2.m:

例如：

input:3
 
output:25
 
我先觀察對角線和平方數之間的關係(找出最接近對角線的值的平方數，然後把平方數扣掉平方數開根號的值減1)，
來算出對角線的總和，然後判斷n是偶數還是奇數後，找出反對角線和平方數之間的關係並相加。

反對角線和平方數之間的關係:反對角線位於矩陣的右上角和左下角，不論是奇數還是偶數，會發現其中一邊是平方數，另一邊是平方數加1。
